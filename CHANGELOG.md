# 更新日志

## 1.1.0-rc.4

- 修复部分教室查询不到课程的问题，查询参数与官网请求保持一致。
- 合并同一教室、同一时间的重复课程记录。
- 教室课表不再展示冗长的教学班名单，只保留课程和教师。

## 1.1.0-rc.3

- 班级课表和教室课表改为与个人课表一致的七天节次网格。
- 修复周次位图直接显示为二进制字符串的问题。
- 移除教室名称中的数字代码占位符，仅展示有效名称。
- 公共课表只展示当前所选周实际开课的课程。

## 1.1.0-rc.2

- 修复空教室查询未传教室编号导致的 EMAP 业务错误。
- 教室列表改为分页获取，降低 Android 端连接被提前关闭的概率。
- 教务系统请求遇到连接中断时自动重试一次。
- GitHub Release 和应用内更新现在会展示本版本更新公告。

## 1.1.0-rc.1

- 成绩页面增加平时、期中、期末、实验等分项成绩。
- 增加空教室、班级课表和教室课表查询入口。
